# 💻 Spring Boot 기반 회원 관리 과제 (Web08)

> 기본적인 회원가입, 로그인/로그아웃, 회원 목록 조회 기능을 구현한 BackEnd과제입니다.

 ## 🔑 주요 기능

1.  **회원가입 (Sign Up):** 사용자 정보 등록 기능
2.  **로그인/로그아웃 (Auth):** 세션 기반 인증 구현
3.  **ID 기억하기:** LocalStorage를 활용한 ID 자동 채움 기능
4.  **회원 목록 조회:** 로그인된 사용자만 접근 가능하도록 권한 처리
5.  **회원 상세 정보 조회:** 목록에서 ID 클릭 시 상세 정보 페이지로 이동
